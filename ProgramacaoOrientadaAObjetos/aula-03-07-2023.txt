PROGRAMAÇÃO ORIENTADA A OBJETOS

    O que é?
        - é um paradigma de análise, projeto, e programação de sistemas de software baseado na composição e interação entre diversas
        unidades de software;

    Para que foi criado?
        - a POO foi criada para tentar aproximar o mundo real do mundo virtual: a ideia é simular o mundo real dentro do computador;

    Para que serve?
        - o Mundo real é extremamento complexo. A orientação a objetos tenta gerenciar a complexidade inerente aos problemas do mundo
        real abstraindo o conhecimanto relevante e encapsulando-o dentro de objetos;

HOSTÓRIA

    A programação Orientada a Objetos foi criada por ALAN KAY, autor da linguagem Smalltalk;

ITENS DE POO

    - Classes - são as caracteristicas de como construir um objeto
    - Objetos - são individuos diferentes dentro de uma classe
    - Atributos - são caracteristicas de um objeto
    - Métodos - são funções que não retornam um resultado
    - Construtores - é o metodo de construção, que garante a sequencia de execução;

CLASSE

    - Toda classe possui um nome;
    - Possui visibilidade, ex: public, private, protected;
    - Possui membros como: Caracteristicas e Ações;
    - Para criar uma classe basta declarar a visibilidade, digitar a palavra reservada class NomeDaClasse e abrir e fechar chaves {}.

    /*
    * public class Teste{
    *     // ATRIBUTOS OU PROPRIEDADES
    *     // MÉTODOS
    * }
    */

OBJETOS

    - São caracteristicas definidas pela classe;
    - São pequenos programas que guardam dentro de si os dados;
    - Trazem as sub-rotinas e as instruções para processar esses dados;

    Objeto: casa
    Classe: planta da casa
